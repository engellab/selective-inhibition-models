Python functions and scripts for the 4 variable decision making model

abc_good_paramsXX--XX.mat model parameters save as a dict

four_var_model.py contains base functions for the model

example_point.py finds fixed points and runs a trial for a set of parameters

run_fourD_scan.py searches selectivity space for fixed points

fvm_gen_psychometrics.py runs simulations to make psychometric, chronometric, and trial completion curves

plot_fvm_examples.m is a Matlab script to plot example fixed points and trial.
