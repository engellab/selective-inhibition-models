Python functions and scripts to train and run RNNs

ei_rnn.py Base library functions and classes

2afc_train_separate_layers.py train a single network, takes paths to params and write directory as arguments 

parameters_seplay.py parameters for a single training

seplay_reload_perturb_current.py runs a perturbation experiment, takes path to trained RNN and path to perturb options file

perturb_curr_opt.py perturbation options
